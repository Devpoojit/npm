Restful Endpoint Structure

http://localhost:9999/restfulservices/v1/users/{id}

1.  http            : Protocol
2.  localhost       :   Host / Domain name
3.  9999            : Port Number
4.  restfulservices :   Application Context
5.  v1              :   Version
6.  users           :   Resource
7.  id              :   Parameter. ID of resource
